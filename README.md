This repository contains code for replicating the results of the Tree of Thoughts paper (NeurIPS 2023). The group members are Joshua McMahon and Kevin Bao.

Sources Used:
1. Tree of Thoughts (https://github.com/princeton-nlp/tree-of-thought-llm)
2. 4x4 Sudoku Dataset (https://github.com/Black-Phoenix/4x4-Sudoku-Dataset)
